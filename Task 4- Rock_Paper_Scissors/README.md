#  Rock Paper Scissors Game

##  Introduction
The Rock Paper Scissors Game is a Python-based game where the user competes against the computer. The computer randomly selects rock, paper, or scissors, and the game determines the winner based on predefined rules.

This project demonstrates randomization, game logic, and interactive Python programming.

---

#  Objectives
- Create an interactive game
- Implement game logic
- Use random module in Python

---

#  Features
- User vs Computer Gameplay  
- Random Computer Choices  
- Winner Determination  
- Score Tracking  
- Multiple Rounds Support  
- Play Again Option
